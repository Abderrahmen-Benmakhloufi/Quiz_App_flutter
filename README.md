# 📱 Flutter Mastery Quiz
A clean and responsive mobile application built with Flutter to help developers practice and master core Flutter concepts.

# 🎯 Project Goals
The goal of this project is to provide a self-assessment tool for Flutter beginners. It covers topics such as:

## Widget Basics:
 Understanding the difference between Stateless and Stateful widgets.

## UI Architecture: 
How Flutter builds and updates the user interface.

## State Management:
 Using setState() to handle dynamic data changes.

# ✨ Key Features
Dynamic Questioning: Questions are pulled from a central data file for easy updates.

Randomized Experience: Answer choices are shuffled for every question to prevent memorization.

**Progress Tracking**: The app records user selections and displays a final summary of correct vs. incorrect answers.

**Themed UI**: Uses Google Fonts and a custom gradient background for a modern developer aesthetic.

## 🛠️ Technical Details
**Custom Widgets**: Includes reusable components like AnswerButton.

**Callback Logic**: Passes data from child widgets to the main parent state via function pointers.

**Logic Operations**: Uses .map(), .where(), and .shuffle() for efficient data manipulation.
